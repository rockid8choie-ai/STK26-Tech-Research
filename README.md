# STK26-Tech-Research

Smart Tech Korea 2026 컨퍼런스에서 수집한 **기술 리서치 아카이브**입니다.
강연 노트, 벤더 발표 요약, 기술·산업 트렌드 정리를 목적으로 합니다.

> 이 저장소는 공개적으로 공유 가능한 컨퍼런스 학습 자료만 담습니다.
> 특정 기업의 제품 전략·사업 계획·내부 자료는 포함하지 않습니다.

## 구성

```
STK26-Tech-Research/
├── Conferences/
│   └── SmartTechKorea_2026/
│       ├── AI_and_Agents/          # 기업용 AI · 에이전트 구조/보안/인증 · SCM 에이전트
│       ├── Quantum/                # 양자컴퓨팅(Braket·NORMA) · 양자보안(IDQ)
│       ├── Robotics_and_Vision/    # 스테레오 비전 · 초소형 정밀로봇
│       ├── Logistics_and_PhysicalAI/  # 물류 Physical AI · 디지털 트윈
│       ├── Manufacturing/          # 제조 XAI · 3D 스캐닝 · 포토닉스
│       └── Innovation_Ecosystem/   # 퀘벡 혁신·투자 생태계
├── Companies/             # 벤더별 요약 (작성 예정)
├── Trend_Reports/         # 테마별 트렌드 종합 (작성 예정)
└── Reference_Materials/   # 용어집·외부 표준 링크 (작성 예정)
```
